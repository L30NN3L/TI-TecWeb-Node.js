Como realizar testes no Projeto

Para ajudar na integração e na elaboração do projeto o mais rápido possível, segue as instruções abaixo:


	1. Caso não tiver Git instalado em sua maquina, baixe e instale: https://git-scm.com/
	2. Veja tutoriais sobre Git no Youtube ou na Udemy, é de graça: https://www.udemy.com/git-e-github-para-iniciantes
	3. Linke seu git remotamente com meu repositório: https://github.com/Leonn3s/Trabalhos.git
	4. Baixe na linha de comando
	5. Caso não tenha Node instalado, é recomendado, só visitar o link e baixar e instalar a versão mais estável: https://nodejs.org/en/
	6. Abra o Menu Iniciar e procure por: “Node.js command prompt”
	7. Vá até ao diretório raiz do projeto pela linha de comando
	8. Digite “nodemon app.js”
	9. Abra o navegador e digite: “localhost:3000”
	10. Após tudo isso, caso não for possível visualizar o Projeto, avise no zap.
	11. Testem todas as funcionalidades possíveis
	12. Testem acompanhando cada notificação que aparecer no prompt de comando do Node.js
	13. Ao encontrar erros digam no zap, ou caso for necessário anotar em um arquivo aqui na nuvem.
	14. Caso tiverem mais dúvidas perguntem no grupo do zap.
	
	PS: Se quiser eu zipo e mando na hora se achar isso muito difícil!
