Como realizar testes no Projeto

Para ajudar na integração e na elaboração do projeto o mais rápido possível, segue as instruções abaixo:


	1. Caso não tiver Git instalado em sua maquina, baixe e instale: https://git-scm.com/
	2. Veja tutoriais sobre Git no Youtube ou na Udemy, é de graça: https://www.udemy.com/git-e-github-para-iniciantes
	3. Procure no Menu Iniciar um arquivo chamado Git Bash, execute
	4. Navegue pela linha de comando e localize uma pasta adequada para baixar o Projeto
	5. Digite na linha de comando "git clone https://github.com/Leonn3s/Trabalhos.git"
	6. Caso não tenha Node instalado, é recomendado, só visitar o link e baixar e instalar a versão mais estável: https://nodejs.org/en/
	7. Abra o Menu Iniciar e procure por: “Node.js command prompt”
	8. Vá até ao diretório raiz do projeto pela linha de comando
	9. Digite “nodemon app.js”
	10. Abra o navegador e digite: “localhost:3000”
	11. Após tudo isso, caso não for possível visualizar o Projeto, avise no zap.
	12. Testem todas as funcionalidades possíveis
	13. Testem acompanhando cada notificação que aparecer no prompt de comando do Node.js
	14. Ao encontrar erros digam no zap, ou caso for necessário anotar em um arquivo aqui na nuvem.
	15. Caso tiverem mais dúvidas perguntem no grupo do zap.
